## 📊 策略概述

MACD激进策略（`macd_aggressive`）是专为**波动较大的ETF**设计的高频交易策略，通过快速进出市场来获取短期收益。

### 核心特点

- ✅ **快速响应**：快速MACD参数（8, 17, 5），灵敏捕捉市场变化
- ✅ **频繁交易**：信号数量多，不要求成交量确认
- ✅ **激进过滤**：放宽买入条件，允许在零轴下方买入
- ✅ **灵活止损**：5%止损，10%/20%分批止盈

## 🎯 策略参数详解

### 1. 技术指标参数

```python
# MACD 参数（快速设置）
fast_period = 8    # 快速EMA周期（默认12）
slow_period = 17   # 慢速EMA周期（默认26）
signal_period = 5  # 信号线周期（默认9）
```

**为什么用快速参数？**

- **快速EMA(8)**：更快反应短期价格变化
- **慢速EMA(17)**：缩短趋势判断周期
- **信号线(5)**：更早捕捉金叉死叉信号

**对比标准参数**：
```
标准参数: 12, 26, 9（适合中长期）
激进参数: 8, 17, 5（适合短期波段）
```

### 2. 零轴过滤（放宽）

```python
zero_axis_filter: True          # 启用零轴过滤
require_zero_above: False       # ⚠️ 不要求在零轴上方
```

**激进之处**：
- ✅ 允许在零轴下方买入（抄底）
- ✅ 不等待DIF回到零轴上方
- ❌ 标准策略要求DIF>0才买入

**示例**：
```
标准策略：DIF必须>0才买入
激进策略：DIF<0也可以买入（只要出现金叉）
```

### 3. MA60过滤（宽松）

```python
ma60_filter: True
ma60_tolerance: 0.05  # 5% 容忍度
```

**激进之处**：
- ✅ 价格低于MA60 5%以内也可以买入
- ❌ 标准策略只允许低于MA60 1%

**示例**：
```
MA60 = 3.00
价格 = 2.88（-4%）

激进策略：✅ 允许买入（4% < 5%容忍度）
标准策略：❌ 不允许买入（通常要求-1%以内）
```

### 4. 背离检测（激进）

```python
enable_divergence: True
divergence_confirm: False     # ⚠️ 不需要确认
min_divergence_count: 1        # ⚠️ 只需1次背离
```

**激进之处**：
- ✅ 出现1次背离即可买入（不等待确认）
- ✅ 底背离直接买入，无需其他确认
- ❌ 标准策略要求2次以上背离且需确认

### 5. 成交量确认（禁用）

```python
volume_confirm: False          # ⚠️ 不要求成交量确认
volume_increase_min: 0.2
volume_increase_max: 0.8
```

**激进之处**：
- ✅ 金叉即可买入，不要求放量
- ❌ 标准策略要求成交量增加30%-50%

### 6. 形态识别（全部启用）

```python
duck_bill_enable: True         # 鸭嘴形态
inverted_duck_enable: True     # 倒鸭嘴形态
```

### 买入规则

1. **金叉买入**：DIF上穿DEA
   - 不要求在零轴上方
   - 不要求成交量放大
   - 不要求价格高于MA60

2. **底背离买入**：价格新低但DIF未创新低
   - 只需1次背离
   - 无需确认

3. **鸭嘴买入**：特殊形态识别

### 卖出规则

#### 止损止盈

```python
stop_loss_pct = 0.05      # 5% 止损
take_profit_pct1 = 0.10   # 10% 卖出30%
take_profit_pct2 = 0.20   # 20% 卖出30%
```

**分批止盈示例**：

```
买入价格: 1.00元

+10% (1.10元): 卖出30%仓位 → 获利30元
+20% (1.20元): 卖出30%仓位 → 获利30元
-5%  (0.95元): 全部止损   → 亏损40元
```

> ⚠️ **重要**：实际代码中使用3个止盈级别（10%/20%/35%），卖出比例为30%/30%/40%。详见[仓位控制详解](MACD_AGGRESSIVE_POSITION_RULES.md)。

#### 死叉卖出

- DIF下穿DEA，清仓
- 不考虑盈亏

## 📈 策略优势

### 1. 捕捉更多机会

**场景1：反弹行情**
```
价格从1.20跌至0.90（-25%）
标准策略：等待DIF>0，可能错过反弹
激进策略：在0.95出现金叉即买入，捕捉到0.95→1.05的反弹
```

**场景2：震荡行情**
```
价格在0.95-1.05之间震荡
激进策略：每个波段都能交易，收益累积
标准策略：错过小波段机会
```

### 2. 快速止损

- 5%止损快速斩断亏损
- 不拖延，不让小亏变大亏

### 3. 分批止盈

- 10%先锁定部分利润
- 20%再锁定部分利润
- 剩余仓位博取更大收益

## ⚠️ 策略风险

### 1. 假突破风险

**问题**：在零轴下方金叉可能是"死猫跳"

```
示例：
价格: 1.00 → 0.80 → 0.85 → 0.75
DIF:  -0.02 → -0.05 → -0.03 (金叉，但仍在负值)
激进策略: 在0.85买入
实际: 继续跌至0.75，触发5%止损
```

### 2. 频繁交易成本

- 交易次数多 → 手续费累积
- 每次交易成本0.5%
- 年化交易成本可能达到5-10%

### 3. 回撤较大

- 允许在下跌趋势中买入
- 可能连续多次止损
- 最大回撤可能超过15%

## 🎪 适用场景

### ✅ 适合场景

1. **波动较大的ETF**
   - 行业ETF（芯片、半导体、新能源）
   - 主题ETF（军工、券商）
   - 波动率>30%的ETF

2. **震荡市**
   - 区间震荡，来回波动
   - 趋势不明显
   - 适合高抛低吸

3. **短期交易**
   - 持有周期1-2周
   - 快进快出
   - 不追求长期收益

### ❌ 不适合场景

1. **单边牛市**
   - 买入持有更好
   - 频繁交易减少收益

2. **缓慢下跌**
   - 反弹都是"死猫跳"
   - 连续止损

3. **低波动ETF**
   - 债券ETF
   - 货币ETF
   - 波动率<10%

## 🔧 优化建议

### 1. 调整止损幅度

```python

# 更保守：8%止损（减少假止损）

```

### 2. 调整止盈点位

```python

take_profit_pct1 = 0.05  # 5%卖出30%

# 慢速止盈（博取更大收益）
take_profit_pct1 = 0.15  # 15%卖出30%

```

```python

```

这样可以避免在下跌趋势中抄底被套。

## 📝 总结

MACD激进策略是一个**高频交易策略**，核心是：

1. **快速参数**（8, 17, 5）→ 灵敏反应
2. **宽松过滤**→ 更多交易机会
3. **严格止损**（5%）→ 控制单次亏损
4. **分批止盈**（10%/20%）→ 锁定利润

**最佳实践**：
- ✅ 用于高波动ETF
- ✅ 在震荡市使用
- ✅ 严格止损，不要抗单
- ❌ 不要在单边牛市使用
- ❌ 不要在缓慢下跌市使用
